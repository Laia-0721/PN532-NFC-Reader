# 贡献指南

感谢您考虑为PN532 NFC读卡器项目做出贡献！

## 如何贡献

### 报告问题
如果您发现了bug或有功能建议：
1. 查看现有Issue，避免重复
2. 创建新的Issue
3. 描述清晰的问题或建议
4. 提供复现步骤（如果可能）

### 提交代码
1. Fork项目
2. 创建功能分支
3. 编写代码
4. 添加测试
5. 提交Pull Request

### 代码规范
- 使用C++17标准
- 遵循现有代码风格
- 添加必要的注释
- 确保代码可读性

### Pull Request流程
1. 从main分支创建新分支
2. 提交清晰的提交信息
3. 确保所有测试通过
4. 更新相关文档
5. 创建Pull Request

## 开发环境设置
参考 [INSTALL.md](INSTALL.md) 设置开发环境。

## 测试
提交代码前请确保：
- 编译通过
- 基本功能正常
- 不破坏现有功能

## 文档
如果您添加了新功能，请同时更新：
- README.md
- 相关使用文档
- 注释文档
