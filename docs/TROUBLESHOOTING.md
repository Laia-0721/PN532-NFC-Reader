# 故障排除指南

## 常见问题

### 1. 程序无法检测到读卡器
**症状**：程序卡在"等待检测"，设备管理器显示"通信串口1"

**解决方案**：
1. 安装CH340驱动
2. 重启电脑
3. 更换USB端口
4. 检查读卡器是否正常供电

### 2. 卡片无法识别
**症状**：放置卡片很久都无法检测

**解决方案**：
1. 调整卡片与读卡器的距离（1-3cm最佳）
2. 远离其他电子设备干扰
3. 尝试不同的卡片
4. 检查读卡器天线是否完好

### 3. 写入失败
**症状**：写入操作失败，返回错误代码

**解决方案**：
1. 确保卡片支持Mifare Classic
2. 检查扇区密钥是否正确
3. 不要写入控制块（除非你知道在做什么）
4. 先备份原始数据

### 4. 编译错误
**症状**：编译时出现各种错误

**解决方案**：
1. 确保安装Visual Studio 2022或MinGW-w64
2. 检查C++标准设置为C++17或更高
3. 安装必要的Windows SDK
